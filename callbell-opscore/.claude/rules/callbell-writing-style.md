# Writing Style When Creating Files

Applies to every file the agent creates or writes (Markdown, text, Excel, PDF, and so on), whatever the
format. The chat reply is not affected.

- **No em dash (—) or en dash (–) as punctuation.** Build sentences from the start so that none is
  needed (period, comma, colon, parenthesis). Sentences should sound natural and human.
- **German output with real umlauts.** When German is expected, write ä, ö, ü, and ß directly, not
  transcribed as ae, oe, ue, or ss.
