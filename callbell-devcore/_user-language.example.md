# User language (yours)

Your personal language for working with the agent. This file is gitignored: copy it to
`_user-language.md` and set your two values. It changes how the agent talks to YOU, not the language
of content written into the repo. That content language stays project-governed and the same for
everyone (see `repo.md` and the area rules), so shared output does not vary by who is editing.

reply: English      # the language the agent replies to you in (chat)
thinking: English   # the language of the agent's visible reasoning while it works
