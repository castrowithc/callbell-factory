# callbell project

This folder is a ready-to-use project template, pre-wired for agentic work with Claude Code and OpenAI
Codex. Copy it, open it in your agent, and run the one-time setup.

## Get started

Run the onboarding once, in your agent's chat:

- Claude Code: `/callbell-onboarding`
- Codex: `@callbell-onboarding`

It sets your language, learns what the project is about, and lays down the rest. You do not have to
configure anything by hand first.

## What is already inside

- Rules and skills for both agents (the `callbell` family; run `/callbell-help` for the list).
- A managed layer `__callbell__/` for context, memory, templates, and the work trail.
- Session context loads automatically at start, so the root files stay minimal on purpose.

That is all you need to begin. Everything else is shaped together with you during onboarding.
